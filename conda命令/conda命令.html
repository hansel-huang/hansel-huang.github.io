主页/conda命令

1、查看当前拥有的虚拟环境：
conda info --envs
或者
conda env list

2、创建虚拟环境（环境名为mmyolo）：
conda create -n mmyolo python=3.8

3、进入虚拟环境：
activate mmyolo

4、查看当前环境中所有安装过的程序：
conda list

5、查看当前环境中所有python的模块：
pip list
出现Could not fetch URL https://pypi.tuna.tsinghua.edu.cn/simple/pip/……是由于SSL证书认证问题，改成以下命令可避免：
pip list --trusted-host pypi.tuna.tsinghua.edu.cn

6、配置镜像提高下载速度（以opencv为例）：
conda配置镜像（永久添加）：
conda config --add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/free/
conda config --add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/main/
conda config --add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/cloud/conda-forge/
conda配置镜像（单条命令）：
conda install opencv -c https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/main/
pip配置镜像：
pip install opencv -i https://mirrors.aliyun.com/pypi/simple/
以下为常用的pip安装镜像：
清华：https://pypi.tuna.tsinghua.edu.cn/simple/
阿里云：https://mirrors.aliyun.com/pypi/simple/
中国科技大学: https://pypi.mirrors.ustc.edu.cn/simple/
华中理工大学：https://pypi.hustunique.com/
山东理工大学：https://pypi.sdutlinux.org/
豆瓣：https://pypi.douban.com/simple/

7、退出环境：
conda deactivate

8、删除环境：
conda remove -n mmyolo --all

9、导出和导入环境：
导出环境：
conda env export > C:/Users/15645/Downloads/environment.yml
导入环境：
conda env create -f /root/autodl-fs/environment.yml
